# 项目模块划分说明

## 概述

本项目已按照功能职责进行模块化划分，所有源代码已移动到 `ResourceCode` 目录下。

**位置**: `ResourceCode/Core/`

**包含文件**:
- `main.cpp` - 应用程序入口
- `Widget.cpp/h/ui` - 主窗口组件

**功能说明**: 
- 应用程序启动和初始化

## 编译说明

项目使用 CMake 构建系统，编译前请确保：

1. Qt6 或 Qt5 开发环境已正确配置
2. MinGW 或 MSVC 编译器可用
3. SQLite3 库已安装
4. libcurl 库已安装（用于天气API）

编译命令：
```bash
cd scripts
./build.bat
```

## 扩展建议

### 添加新模块
1. 在 `ResourceCode/` 下创建新模块目录
2. 将相关源码移动到新目录
3. 更新 `CMakeLists.txt` 中的 `PROJECT_SOURCES` 变量
4. 在主窗口或其他模块中引入新模块的头文件

### 模块独立性
建议在未来重构中考虑：
- 使用 Qt 的库系统（`add_library`）将每个模块编译为独立的静态库
- 定义清晰的模块接口（头文件）
- 减少模块间耦合，提高代码可维护性
