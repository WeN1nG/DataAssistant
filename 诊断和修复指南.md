### 常见问题

#### 问题1：代码没有正确编译

**症状**：修改代码后，功能没有变化

**解决**：
1. 清理项目：Build → Clean All
2. 重新编译：Build → Rebuild All
3. 关闭 Qt Creator
4. 删除 build 目录（如果有）
5. 重新打开项目并编译

#### 问题2：数据库数据问题

**症状**：删除操作看起来成功，但数据没变

**解决**：
```sql
-- 检查实际数据
SELECT * FROM schedules WHERE isBatch = 1;

-- 如果有残留数据，手动删除
DELETE FROM schedules WHERE isBatch = 1;
```

**解决**：
1. 添加日志输出
