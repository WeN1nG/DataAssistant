## 三、代码逻辑说明

### 3.1 日期判断优先级顺序

```
1. 是否是今天(isToday)?
   ├─ 是 → 白色背景 + 加粗字体 + 优先级颜色覆盖
   └─ 否 → 进入下一步

2. 是否是当月已过去日期(isPastDate)?
   ├─ 是 → 浅灰色背景(#F5F5F5) + 优先级颜色覆盖
   └─ 否 → 进入下一步

3. 是否是非当月日期(!isCurrentMonth)?
   ├─ 是 → 深灰色背景(#E0E0E0)
   └─ 否 → 进入下一步

4. 其他（当月未来日期）
   └─ 白色背景(#FFFFFF) + 优先级颜色覆盖
```

### 3.2 关键代码片段

```cpp
// 日期判断逻辑
bool isToday = (date == m_today);
bool isPastDate = isCurrentMonth && (date < m_today) && !isToday;
bool isCurrentMonth = (date.month() == m_currentMonth);
int dayOfWeek = date.dayOfWeek();

// 背景颜色设置
if (isToday) {
    // 白色背景，除非有日程
    if (maxPriority >= 0) {
        // 显示优先级颜色
    } else {
        backgroundColor = QColor(255, 255, 255);  // 白色
    }
} else if (isPastDate) {
    // 浅灰色背景
    backgroundColor = QColor(245, 245, 245);
} else if (!isCurrentMonth) {
    // 深灰色背景
    backgroundColor = QColor(224, 224, 224);
} else {
    // 白色背景
    backgroundColor = QColor(255, 255, 255);
}

// 字体设置
font.setBold(isToday);  // 只有今天是加粗

// 周末红色字体
if (dayOfWeek == 6 || dayOfWeek == 7) {
    painter->setPen(QColor(255, 0, 0));  // 红色
}
```

### 4.2 颜色层次

1. **白色层**（最高优先级）: 当月未来日期、当日（无日程）
2. **浅灰色层**: 当月已过去日期（无日程）- #F5F5F5
3. **深灰色层**: 非当月日期 - #E0E0E0
4. **优先级色层**: 任意日期类型（有日程时覆盖基础色）
