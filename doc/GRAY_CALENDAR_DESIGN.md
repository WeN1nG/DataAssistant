# 日历灰色配色方案设计文档

## 设计目标

使用统一的灰色配色方案设计日历组件，确保：
- 符合WCAG无障碍标准（对比度 ≥ 4.5:1）
- 在所有视图模式下保持一致
- 适配不同屏幕尺寸

---

## 配色方案

### 核心颜色

| 颜色名称 | 色值 | RGB | 用途 |
|---------|------|-----|------|
| **深灰色** | `#333333` | RGB(51, 51, 51) | 文字、边框、选中状态 |
| **浅灰色** | `#F5F5F5` | RGB(245, 245, 245) | 背景、单元格底色 |

---

## WCAG 对比度分析

### 1. 文字与背景对比度

**深灰色文字 (#333333) 在浅灰色背景 (#F5F5F5) 上**

计算步骤：

1. **深灰色 #333333 的相对亮度**：
   - RGB: (51, 51, 51)
   - 归一化: (0.2, 0.2, 0.2)
   - 线性化: (0.02, 0.02, 0.02)  [因为 0.2 < 0.04045]
   - **相对亮度 L1 = 0.016**

2. **浅灰色 #F5F5F5 的相对亮度**：
   - RGB: (245, 245, 245)
   - 归一化: (0.96, 0.96, 0.96)
   - 线性化: (0.92, 0.92, 0.92)  [因为 0.96 > 0.04045]
   - **相对亮度 L2 = 0.92**

3. **对比度计算**：
   ```
   对比度 = (L1 + 0.05) / (L2 + 0.05)
          = (0.92 + 0.05) / (0.016 + 0.05)
          = 0.97 / 0.066
          = 14.7 : 1
   ```

### 2. WCAG 标准对比

| WCAG 级别 | 最低对比度要求 | 实际对比度 | 状态 |
|----------|--------------|-----------|------|
| AA 级（普通文本） | **4.5:1** | **14.7:1** | ✅ **通过** |
| AAA 级（增强文本） | **7:1** | **14.7:1** | ✅ **通过** |

**结论**：深灰色 #333333 与浅灰色 #F5F5F5 的对比度为 **14.7:1**，远超 WCAG 所有级别的要求！

## 配色应用规则

### 1. 日历单元格

| 元素 | 颜色 | 对比度 | 用途 |
|------|------|--------|------|
| 普通日期文字 | #333333 | 14.7:1 | 清晰可读 |
| 日期背景 | #F5F5F5 | - | 浅色底色 |
| 周末日期 | #333333 | 14.7:1 | 与普通日期一致 |
| 非当月日期 | #333333 (50% 透明度) | ~7:1 | 降低对比度表示次要 |

### 2. 选中状态

| 元素 | 颜色 | 说明 |
|------|------|------|
| 选中日期背景 | #333333 (20% 透明度) | 深灰色半透明 |
| 选中日期文字 | #333333 | 保持高对比度 |
| 选中边框 | #333333 | 深灰色边框 |

### 3. 悬停状态

| 元素 | 颜色 | 说明 |
|------|------|------|
| 悬停日期背景 | #333333 (10% 透明度) | 更浅的深灰色 |
| 悬停日期文字 | #333333 | 保持清晰 |

### 4. 今日标记

| 元素 | 颜色 | 说明 |
|------|------|------|
| 今日背景 | #333333 | 深灰色实心 |
| 今日文字 | #F5F5F5 | 浅色文字确保可读 |

## 样式表实现

### QCalendarWidget 样式表

```css
QCalendarWidget {
    /* 日历整体背景 */
    background-color: #F5F5F5;
    border: 1px solid #333333;
}

/* 导航栏（月份选择器） */
QCalendarWidget QWidget#qt_calendar_navigationbar {
    background-color: #333333;
    color: #F5F5F5;
}

/* 月份年份标签 */
QLabel {
    color: #F5F5F5;
}

/* 日期表格视图 */
QCalendarWidget QAbstractItemView {
    background-color: #F5F5F5;
    border: none;
    color: #333333;
    selection-background-color: rgba(51, 51, 51, 50);
    selection-color: #333333;
}

/* 普通日期 */
QCalendarWidget QAbstractItemView::item {
    background-color: #F5F5F5;
    color: #333333;
}

/* 悬停状态 */
QCalendarWidget QAbstractItemView::item:hover {
    background-color: rgba(51, 51, 51, 25);
    color: #333333;
}

/* 选中状态 */
QCalendarWidget QAbstractItemView::item:selected {
    background-color: rgba(51, 51, 51, 50);
    color: #333333;
    border: 1px solid #333333;
}

/* 周末日期（保持一致） */
QCalendarWidget QAbstractItemView QModelIndex {
    color: #333333;
}
```

## 不同视图模式的适配

### 月视图 (Month View)

- 7列 x 6行网格
- 每个单元格大小均匀分配
- 颜色方案：**深灰色文字 + 浅灰色背景**

### 周视图 (Week View)

- 7列 x 1行显示
- 时间轴与日期分离
- 颜色方案：与月视图保持一致

### 日视图 (Day View)

- 单日详细视图
- 时间轴使用深灰色
- 背景使用浅灰色

```css
/* 基础字体大小 */
QCalendarWidget QAbstractItemView {
    font-size: 14px;
}

/* 小屏幕 */
@media (max-width: 480px) {
    QCalendarWidget QAbstractItemView {
        font-size: 12px;
    }
}
```

## 实施注意事项

1. **保持简洁**：只使用两种灰色，避免渐变或额外颜色
2. **测试对比度**：使用专业工具验证实际对比度
3. **跨浏览器/平台测试**：确保 Qt 样式表在所有平台上表现一致
4. **无障碍测试**：使用屏幕阅读器验证可用性
