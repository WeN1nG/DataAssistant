# 今天和背景染色冲突问题修复

## 问题描述

当"今天"（有黑色边框高亮）和"今天有日程"（有背景染色）同时发生时，背景染色失效，显示为统一的浅灰色背景。

## 问题原因

在原有的 paint() 方法中，判断顺序是：

```cpp
if (isToday) {
    // 问题：这里直接使用固定背景色，忽略了优先级
    backgroundColor = QColor(245, 245, 245);
    textColor = QColor(51, 51, 51);
    font.setBold(true);
    drawBorder = true;
} else if (maxPriority == 2) {
    // 这段代码永远不会执行，因为 isToday 已经先匹配了
    backgroundColor = QColor(255, 180, 180);
    // ...
}
```

**问题**：当某天既是今天又有紧急日程时，只匹配了 `isToday` 条件，背景色被固定为浅灰色，优先级颜色被忽略。

## 解决方案

修改 isToday 的处理逻辑，使其在今天是日程日期时也能显示优先级颜色：

```cpp
if (isToday) {
    // 今天需要显示边框，同时保留优先级颜色
    font.setBold(true);
    drawBorder = true;
    
    // 今天的背景色根据优先级变化
    if (maxPriority == 2) {
        // 紧急：淡红色背景
        backgroundColor = QColor(255, 180, 180);
        textColor = QColor(51, 51, 51);
    } else if (maxPriority == 1) {
        // 重要：淡黄色背景
        backgroundColor = QColor(255, 255, 180);
        textColor = QColor(51, 51, 51);
    } else if (maxPriority == 0) {
        // 一般：淡蓝色背景
        backgroundColor = QColor(180, 200, 255);
        textColor = QColor(51, 51, 51);
    } else {
        // 无日程：使用默认的浅灰色
        backgroundColor = QColor(245, 245, 245);
        textColor = QColor(51, 51, 51);
    }
}
```

## 修改后的效果

### 场景2：今天是重要日程

### 场景3：今天是一般日程

- **修改前**：浅灰色背景 + 黑色边框（忽略优先级）
- **修改后**：淡蓝色背景 + 黑色边框 ✓

### 场景4：今天无日程

- **修改前**：浅灰色背景 + 黑色边框 ✓
- **修改后**：浅灰色背景 + 黑色边框 ✓（保持不变）

## 技术实现要点

### 1. 优先级判断优先

在 `isToday` 分支内部，再次根据 `maxPriority` 设置不同的背景色，确保今天有日程时能正确显示优先级颜色。

### 2. 边框始终显示

无论今天的日程优先级如何，黑色边框始终显示，确保"今天"的高亮效果不受影响。

### 3. 文字样式一致

今天的文字始终为加粗，但背景色会根据优先级变化，保持视觉层次。

## 完整的判断逻辑

```cpp
if (isToday) {
    // 今天需要显示边框，同时保留优先级颜色
    font.setBold(true);
    drawBorder = true;
    
    // 今天的背景色根据优先级变化
    if (maxPriority == 2) {
        backgroundColor = QColor(255, 180, 180);  // 淡红色
    } else if (maxPriority == 1) {
        backgroundColor = QColor(255, 255, 180);  // 淡黄色
    } else if (maxPriority == 0) {
        backgroundColor = QColor(180, 200, 255);  // 淡蓝色
    } else {
        backgroundColor = QColor(245, 245, 245);  // 浅灰色
    }
} else if (!isCurrentMonth) {
    // 非当月日期
    backgroundColor = QColor(238, 238, 238);
} else if (maxPriority == 2) {
    // 紧急
    backgroundColor = QColor(255, 180, 180);
} else if (maxPriority == 1) {
    // 重要
    backgroundColor = QColor(255, 255, 180);
} else if (maxPriority == 0) {
    // 一般
    backgroundColor = QColor(180, 200, 255);
} else {
    // 无日程
    backgroundColor = QColor(250, 250, 250);
}
```
