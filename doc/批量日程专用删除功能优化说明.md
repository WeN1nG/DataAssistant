# 批量日程专用删除功能优化说明

## 优化日期
2026-04-07

## 问题背景

### 原始问题
系统中的批量日程删除功能存在异常行为：
- **问题描述**：系统提示将删除全部日程，但实际仅删除单天日程
- **影响范围**：批量删除选中日程、一键清空所有日程
- **用户体验**：删除确认信息不准确，无法了解实际删除范围

### 问题根源
之前的实现只统计**删除的批次数**（调用了几次），而不是**实际删除的日程总数**（包括批量日程中的每一天）。

```cpp
// 问题代码示例
for (int id : ids) {
    deleteBatch(id);  // 删除整个批次（可能是5天）
    deletedCount++;  // ❌ 只计数为1，而不是5
}
```

---

## 解决方案

### 核心改进
开发了**专用的批量日程删除函数**，能够：
1. ✅ 准确统计实际删除的日程总数
2. ✅ 区分批量日程批次数和实际天数
3. ✅ 展示详细的删除范围预览
4. ✅ 提供清晰的删除结果反馈

---

## 新增功能

### 1. DeleteResult 数据结构

```cpp
struct DeleteResult {
    int totalSchedules;      // 将要/实际删除的日程总数（核心指标）
    int batchCount;         // 涉及的批量日程批次数
    int scheduleCount;      // 涉及的批量日程天数（或普通日程数）
    int actualDeleted;      // 实际删除的日程数
    QStringList batchDetails; // 每个批量日程的详细信息
};
```

### 2. getDeletePreview() - 删除预览函数

**功能**：在执行删除前，预览将要删除的日程详情

**参数**：
- `ids`: 要删除的日程ID列表

**返回值**：
- `DeleteResult`: 包含详细统计信息

**使用场景**：
- 在显示确认对话框前，向用户展示准确的删除信息
- 帮助用户决定是否执行删除操作

**代码示例**：
```cpp
QVector<int> ids = {1, 2, 3, 4, 5};
DeleteResult preview = dbManager->getDeletePreview(ids);

// 预览结果示例
// preview.totalSchedules = 10  // 实际10天
// preview.batchCount = 2        // 2批批量日程
// preview.scheduleCount = 8     // 批量日程共8天
// preview.batchDetails = ["项目A (5天: 04/01 至 04/05)", "会议B (3天: 04/10 至 04/12)"]
```

---

### 3. deleteSchedulesWithDetails() - 详细删除函数

**功能**：执行批量删除，并返回详细的删除结果

**参数**：
- `ids`: 要删除的日程ID列表

**返回值**：
- `DeleteResult`: 包含实际删除的详细信息

**特性**：
- ✅ 使用事务机制保证原子性
- ✅ 避免重复删除同一批次
- ✅ 准确统计实际删除数量
- ✅ 返回详细的删除报告

**代码示例**：
```cpp
QVector<int> ids = {1, 2, 3, 4, 5};
DeleteResult result = dbManager->deleteSchedulesWithDetails(ids);

qDebug() << "实际删除了" << result.actualDeleted << "个日程";
```

## 实现细节

### 1. 避免重复处理机制

使用 `QSet<QString>` 跟踪已处理的批量日程批次：

```cpp
QSet<QString> processedBatches;  // 用于避免重复计算同一批次

for (int id : ids) {
    QString batchKey = QString("%1|%2|%3|%4")
        .arg(title)
        .arg(datetime)
        .arg(endDatetime)
        .arg(priority);
    
    if (!processedBatches.contains(batchKey)) {
        processedBatches.insert(batchKey);
        // 处理这个批次
    }
}
```

### 2. 天数计算

```cpp
QDateTime startDate = QDateTime::fromString(datetime, Qt::ISODate);
QDateTime endDate = QDateTime::fromString(endDatetime, Qt::ISODate);
int days = startDate.date().daysTo(endDate.date()) + 1;
```

### 3. 事务保证

```cpp
db.transaction();
for (int id : ids) {
    // 查询和删除逻辑
}
db.commit();  // 原子性保证
```

struct DeleteResult {
    int totalSchedules;      // 将要/实际删除的日程总数
    int batchCount;         // 涉及的批量日程批次数
    int scheduleCount;      // 涉及的批量日程天数（或普通日程数）

### ScheduleListDialog.cpp 改进

**onBatchDelete() 改进**：
```cpp
void ScheduleListDialog::onBatchDelete()
{
    // 1. 获取删除预览
    DeleteResult preview = dbManager->getDeletePreview(ids);
    
    // 2. 显示详细确认信息
    QString message = QString("📊 共计: %1 个日程\n").arg(preview.totalSchedules);
    message += QString("📅 批量日程: %1 批（共 %2 天）\n").arg(preview.batchCount).arg(preview.scheduleCount);
    // ...
    QMessageBox::question(this, "确认删除", message);
    
    // 3. 执行删除
    DeleteResult result = dbManager->deleteSchedulesWithDetails(ids);
    
    // 4. 显示详细结果
    QString successMessage = QString("✅ 已成功删除 %1 个日程\n").arg(result.actualDeleted);
    QMessageBox::information(this, "删除成功", successMessage);
}
```
