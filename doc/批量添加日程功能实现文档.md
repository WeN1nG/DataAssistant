# 批量添加日程功能实现文档

## 功能概述

本文档详细说明了在个人日程助手中实现的批量添加日程功能。该功能允许用户一次性添加多个日期的日程，提高了日程管理的效率。

## 一、功能需求

### 1.1 基本需求

1. **界面设计**：在现有添加日程窗口中新增批量添加勾选框
2. **日期范围选择**：支持选择起点和终点日期
3. **批量保存**：一次性创建多个日程条目
4. **合并显示**：在日程视图中合并显示日期范围

### 1.2 约束条件

- 单次批量添加不超过30天
- 结束日期必须晚于或等于开始日期
- 批量添加的日程应与普通日程有明显区分

```cpp
    bool enabled = (state == Qt::Checked);

    } else {
```

### 4.3 日期验证逻辑

```cpp
bool ScheduleDialog::validateBatchDates() {
    QDateTime startDate = ui->startDateTimeEdit->dateTime();
    QDateTime endDate = ui->endDateTimeEdit->dateTime();
    
    // 验证1：结束日期必须晚于或等于开始日期
    if (endDate < startDate) {
        QMessageBox::warning(this, "日期错误", 
            "结束日期必须晚于或等于开始日期");
        return false;
    }
    
    // 验证2：日期范围不能超过30天
    int days = getDaysInRange(startDate, endDate);
    if (days > 30) {
        QMessageBox::warning(this, "日期范围过大", 
            QString("单次批量添加不能超过30天，当前选择为 %1 天\n"
                    "请缩短日期范围后重试").arg(days));
        return false;
    }
    
    return true;
}
```

### 4.4 批量添加核心逻辑

```cpp
void ScheduleDialog::on_saveButton_clicked() {
    if (ui->batchAddCheckBox->isChecked()) {
        // 批量添加模式
        if (!validateBatchDates()) {
            return;
        }
        
        QDateTime startDate = ui->startDateTimeEdit->dateTime();
        QDateTime endDate = ui->endDateTimeEdit->dateTime();
        
        // 创建批量日程数组
        QVector<Schedule> schedules;
        QDateTime currentDate = startDate;
        
        while (currentDate <= endDate) {
            Schedule schedule;
            schedule.id = -1;
            schedule.completed = false;
            schedule.title = title;
            schedule.description = description;
            schedule.datetime = currentDate;
            schedule.endDatetime = endDate;
            schedule.isBatch = true;
            schedule.priority = priority;
            schedule.color = getColorByPriority(priority);
            
            schedules.append(schedule);
            currentDate = currentDate.addDays(1);
        }
        
        // 使用事务批量添加
        if (dbManager.addBatchSchedules(schedules)) {
            QString message = QString("成功添加 %1 个日程（%2 至 %3）")
                .arg(schedules.size())
                .arg(startDate.toString("yyyy-MM-dd"))
                .arg(endDate.toString("yyyy-MM-dd"));
            QMessageBox::information(this, "成功", message);
            accept();
        }
    }
}
```

```cpp
bool DatabaseManager::addBatchSchedules(const QVector<Schedule>& schedules) {
    if (schedules.isEmpty()) {
        return false;
    }
    
    QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
    db.setDatabaseName(dbPath);
    
    if (!db.open()) {
        return false;
    }
    
    bool success = true;
    db.transaction();  // 开始事务
    
    for (const Schedule& schedule : schedules) {
        // 插入每个日程...
        if (!query.exec()) {
            success = false;
            break;
        }
    }
    
    if (success) {
        db.commit();   // 提交事务
    } else {
        db.rollback(); // 回滚事务
    }
    
    db.close();
    return success;
}
```

### 7.3 事务回滚机制

批量添加使用数据库事务确保原子性：

```cpp
db.transaction();
for (const Schedule& schedule : schedules) {
    if (!query.exec()) {
        db.rollback();  // 任何一个失败，全部回滚
        return false;
    }
}
db.commit();  // 全部成功，提交事务
```

## 十一、性能考虑

### 11.1 批量添加性能

- 使用数据库事务，减少磁盘I/O
- 30天限制防止一次性创建过多日程
- 使用预处理SQL语句，提高执行效率

### 11.2 查询性能

- 日期范围查询使用索引优化
- 支持按日期快速过滤

- 使用 `QVector` 管理日程数组
- 避免不必要的数据复制
