# 日志删除成功提示框移除

## 修改概述

为了保持界面简洁，移除了日程（日志）删除成功后的提示框功能。

## 修改内容

### 1. ScheduleListDialog.cpp - 第 413-414 行

**删除前**:
```cpp
if (dbManager->deleteSchedule(scheduleId)) {
    QMessageBox::information(this, "成功", "日程已删除");
    refreshScheduleList();
    emit accepted();
}
```

**删除后**:
```cpp
if (dbManager->deleteSchedule(scheduleId)) {
    refreshScheduleList();
    emit accepted();
}
```

### 2. ScheduleDetailDialog.cpp - 第 396-397 行

**删除前**:
```cpp
if (dbManager->deleteSchedule(scheduleId)) {
    QMessageBox::information(this, "成功", "日程已删除");
    refreshSchedules();
    emit accepted();
}
```

**删除后**:
```cpp
if (dbManager->deleteSchedule(scheduleId)) {
    refreshSchedules();
    emit accepted();
}
```

## 功能说明

### 修改后的行为

1. **删除成功时**：
   - 直接刷新日程列表（通过 `refreshScheduleList()` 或 `refreshSchedules()`）
   - 发送 `accepted()` 信号通知主窗口刷新日历标记
   - 不显示任何提示框

2. **删除失败时**：
   - 仍然显示错误警告框（`QMessageBox::warning`）
   - 确保用户知道操作失败

## 用户体验优化

- **更简洁的界面**：减少不必要的弹窗
- **即时反馈**：通过列表的实时更新来反馈操作结果
- **保持关键提示**：仅在操作失败时显示警告，保持必要的错误提示功能
