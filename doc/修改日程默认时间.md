# 日程默认时间修改

## 修改概述

将新建日程时的默认时间从"当前时间"改为"今天的8点"。

## 修改内容

### 文件：ScheduleDialog.cpp

**位置**：第 34-36 行

**修改前**：
```cpp
} else {
    // 新建模式，设置默认时间为当前时间
    ui->dateTimeEdit->setDateTime(QDateTime::currentDateTime());
}
```

**修改后**：
```cpp
} else {
    // 新建模式，设置默认时间为今天的8点
    ui->dateTimeEdit->setDateTime(QDateTime(QDate::currentDate(), QTime(8, 0, 0)));
}
```

## 技术说明

### QDateTime 构造函数

```cpp
QDateTime(QDate date, QTime time)
```

- `QDate::currentDate()` - 获取当前日期
- `QTime(8, 0, 0)` - 创建一个8:00:00的时间对象

### 效果

- **日期**：自动设置为今天的日期
- **时间**：固定为 08:00:00
- **用户仍可修改**：用户可以在时间选择器中修改为其他时间

## 使用场景

这个默认值适用于：
- 早晨的日程安排
- 办公时间的开始时间
- 统一的工作计划时间点
